# Инструкция по обновлению деплоя (Cloudflare Pages)

Эта инструкция поможет вам собрать и загрузить обновленную версию сайта в Cloudflare Pages.

## Предварительные требования
Убедитесь, что у вас установлены:
- Node.js и npm
- Cloudflare Wrangler CLI. Если не установлено, мы будем использовать `npx` (см. ниже).

## Пошаговая инструкция

### 1. Сборка проекта (Build)
Перед деплоем необходимо создать оптимизированную production-версию приложения.
Откройте терминал в корневой папке проекта и запустите:

```bash
npm run build
```

Эта команда создаст (или обновит) папку `dist` с готовыми файлами.

### 2. Деплой (Deploy)
Загрузка файлов из папки `dist` в Cloudflare Pages.
Используйте команду:

```bash
npx wrangler pages deploy dist
```

Если вы запускаете это впервые, Wrangler попросит вас авторизоваться в Cloudflare через браузер.
1. Нажмите `Allow` в открывшемся окне браузера.
2. Вернитесь в терминал.
3. Вас могут спросить: `Create a new project?` (если проект еще не создан). Выберите имя проекта, например `diy-moto-anti`.
4. Если проект уже существует, выберите его из списка или введите название.

### 3. Проверка
После успешного деплоя вы увидите ссылку на ваш сайт в терминале (обычно заканчивается на `*.pages.dev` или ваш привязанный домен `diymotogarage.com`).

*Примечание: Обновление DNS и кэша может занять пару минут.*
